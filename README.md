# QuickPlan

Sistema de reservas inteligente para experiencias enoturísticas en Bodegas Otazu.

## Descripción

QuickPlan es una aplicación web que facilita la gestión y reserva de visitas y experiencias en la bodega. Permite a los usuarios explorar diferentes tipos de experiencias, seleccionar fechas y horarios, y completar reservas de forma intuitiva.

## Tecnologías

- React + Vite
- React Router
- CSS

## Instalación y ejecución

1. Instala las dependencias:
```bash
npm install
```

2. Inicia el servidor de desarrollo:
```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`
